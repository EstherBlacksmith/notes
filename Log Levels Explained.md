| Level | When to Use | Example |
|------|-------------|---------|
| TRACE | Very detailed, rarely used | Entry/exit of methods |
| DEBUG | Development info, troubleshooting | "playerHit called", method parameters |
| INFO | Business events, key milestones | "Game started", "Player won", "Game finished" |
| WARN | Potential issues, recoverable errors | "Low disk space", "Retrying operation" |
| ERROR | Failures, exceptions | "Database connection failed", "Validation error" |
